# Evaluation-of-differentiated-contributions
Оценка точности реализации карт заданий для дифференцированного внесения удобрений

Пользователь выбирает три слоя для анализируемого поля:
  Complited - фактическое внесение поля;
  Contour - контур анализируемого поля;
  Task - планируемое внесение;
  
После ползователь выбирает поля из таблицы для анализа ( они должны соответствовать друг-другу ):
  Поле с фактическим значением внесения из Complited (Seed или Fert);
  Поле с плановым внесением из Task (1_seed или 1_fert);
  
Далее идут значения буферов, они заданы по умолчанию, но пользователь может подогнать их под свое поле

На выходе получаем поля с плохим внесением, а также файл txt с общей характеристикой поля
